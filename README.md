Projekt To-Do-List

Prosta aplikacja webowa typu To-Do List, napisana w czystym HTML, CSS oraz JavaScript.
Projekt umożliwia dodawanie, oznaczanie jako wykonane oraz usuwanie zadań.

To klasyczny projekt front-endowy skupiający się na manipulacji DOM oraz obsłudze zdarzeń w JavaScript.

1.Funkcjonalności

-Dodawanie nowych zadań

-Oznaczanie zadań jako wykonane

-Usuwanie zadań z listy

-Prosty, czytelny interfejs użytkownika

2.Technologie

a) Projekt został wykonany bez użycia frameworków i bibliotek zewnętrznych.

-HTML5

-CSS3

-JavaScript

Struktura projektu
Projekt-To-Do-List/
│
├── index.html      # Struktura aplikacji
├── style.css       # Stylowanie interfejsu
├── script.js       # Logika działania aplikacji
└── images/         # grafiki użyte w projekcie

3.Jak działa aplikacja

-Użytkownik wpisuje treść zadania w pole input.

-Po kliknięciu przycisku zadanie zostaje dodane do listy.

-Kliknięcie na zadanie oznacza je jako wykonane (zmiana stylu).

-Przycisk usuwania pozwala skasować zadanie z listy.

-Cała logika działania opiera się na manipulacji elementami DOM oraz obsłudze zdarzeń (click, submit itp.).

4.Możliwe rozszerzenia

a) Projekt można rozbudować o:

-Zapisywanie zadań w localStorage

-Lepszą responsywność

-Sortowanie lub przeciąganie zadań (drag & drop)

5.Cel projektu

a) Projekt został wykonany w celach edukacyjnych jako ćwiczenie:

-pracy z DOM
